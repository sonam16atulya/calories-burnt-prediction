# Calorie Burnt Prediction

## Overview
Developed a machine learning model to predict calories burned during physical activities using user health and exercise-related features.

## Dataset Features
- Age, Gender, Height, Weight
- Duration, Heart Rate, Body Temperature
- Calories Burned

## Methodology
- Performed data preprocessing and exploratory data analysis
- Applied feature engineering and correlation analysis
- Built regression models to estimate calorie expenditure

## Models Used
- Linear Regression

## Evaluation Metrics
- MAE
- RMSE
- R² Score

## Tech Stack
Python | Pandas | NumPy | Scikit-learn| Matplotlib

## Outcome
Built an accurate predictive model to estimate calorie burn and identify key factors affecting energy expenditure.
